Forum Software

I started working on a forum project in my own time in the middle of 2015, learning HTML, PHP and Javascript to encorporate into different parts of the website. The forum was created from scratch and had the ability for:

  • Ability to post and respond to threads in different catagories throughout the forum, with the ability to edit and delete the posts afterwards.
  • Ability to view and customise their own profiles, with others also being able to see the customisations.
  • Ability to edit and update account information such as email address, password and username.

Some of the skills I learnt while working on the project are:

  • Proper usage of PHP elements and how to integrate Javascript.
  • Proper encryption techniques such as hashing and MD5 to allow safe storage of information in an SQL database
  • Decent knowledge of PHP with the ability to apply it to other projects.
Surf Shop

As part of a web-development project for HNC Computing I had to design and code an e-commerce website in the theme of a surfing shop which used HTML, CSS and PHP to allow the user to order items from the store, some of the features of this project were:

  • Ability to add or remove items with different quantities from the cart, with the total price updated.
  • A receipt page upon ordering the products.
  • Database entry for each order, including costs and quantity of item ordered.
  • Encrypting passwords using in-built hashing through PHP.
